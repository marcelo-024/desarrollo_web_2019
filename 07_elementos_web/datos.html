Formularios

Los formularios se incluyen dentro de la etiqueta <form>.

En ellos se utilizan los siguientes controles. El formulario es un interfaz que nos permite interactuar con el usuario.

  • input - text
  • text area
  • checkbox
  • radio butons
  • select / options
  • botones

La etiqueta form tiene dos atributos importantes

Estos definen como funciona nuestro formulario, son:

  • Action
  • Method
